# 📦 RetailIQ — AI-Powered Retail Demand Forecasting System

> **Production-grade** demand forecasting and inventory optimization for retail Beverages & Snacks categories.  
> XGBoost · FastAPI · Streamlit · SQLite · Real-time inventory alerts.

---

## 📊 Model Performance

| Model | MAE | Improvement |
|---|---|---|
| Baseline XGBoost | 4.8769 units | — |
| Improved XGBoost | **4.3601 units** | **↑ 10.6%** |

**Top features discovered:** `rolling_mean_7`, `lag_1`, `promo_intensity`, `cross_product_demand_signal`

**Business insight:** All products need daily replenishment — net_stock covers only ~0.68× daily demand vs the required ~3.4× reorder buffer. This is a structural insight, not a model error.

## 🚀 Quick Start

### 1. Clone & Install

```bash
git clone https://github.com/yourname/retail-forecast.git
cd retail-forecast
pip install -r requirements.txt
```

### 2. Add Your Data

Place `augmented_ml_ready_dataset.csv` in the `data/` folder.

### 3. Train the Model

```bash
python train_model.py --data data/augmented_ml_ready_dataset.csv
```

Output: `backend/models/demand_model.pkl`

### 4. Launch the Dashboard

```bash
streamlit run app.py
```

### 5. Launch the API (optional)

```bash
cd retail_forecast
uvicorn backend.main:app --reload --port 8000
# API docs: http://localhost:8000/docs
```

## 🧠 ML Pipeline

```
Raw Input
    │
    ▼
engineer_features()
    │  category_encoded, price, flags
    │  lag_1, lag_7, rolling_mean_7, rolling_std_7
    │  promo_intensity = promotion_flag × marketing_intensity
    │  high_marketing_flag (binary above-median)
    │  demand_ratio = beverage_demand / (snack_demand + 1)
    │  cross_product_demand_signal (opposite category daily total)
    ▼
XGBoost Regressor (n_estimators=300, max_depth=6, lr=0.05)
    │
    ▼
predict_demand() → float (≥ 0, clipped)
    │
    ▼
compute_inventory_analytics()
    │  safety_stock = Z × σ × √(lead_time)
    │  reorder_point = μ × lead_time + safety_stock
    │  reorder_quantity = target_stock - current_stock
    │  stockout_risk_pct = (1 - stock/reorder_point) × 100
    │  health_score (0–100 composite)
    ▼
generate_recommendation() → List[str]
```
